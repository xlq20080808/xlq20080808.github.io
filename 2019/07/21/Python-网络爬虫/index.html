网络爬虫

定义

  • 网络爬虫：按照一定规则从互联网获取信息的一组程序
  • 通用网络爬虫：通过访问URL路径过程中不断更新其中URL路径集，反复访问获取数据的过程
  • 聚集网络爬虫：访问路径前，通过网页解析算法，对路径进行筛选，筛选后获取所需的网页数据过程
  • 通过互联网以及URL获取网页数据的过程

urlopen

  • 打开路径对应的网页，完成生成file类型的对象，将网页源代码通过read方法进行读取，需要decode解码

request

  • 指定请求路径及传输请求header和data的对象

post数据
