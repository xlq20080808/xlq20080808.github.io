Python-正则表达式

正则表达式

概念

  • 一组文本或特殊符号组成的字符串， 用于检索、验证或交替其他文本

常用符号

功能字符

  • ‘*’表示匹配0位到多位

  • ‘+’表示匹配1位到多位

  • ‘？’表示匹配0位到1位

  • ‘{n}’表示匹配n位

  • ‘^’表示从文本开头进行匹配

  • ‘$’表示结尾匹配

  • ‘^’与’$’结合使用 从开头到结尾完全匹配

  • ‘.’表示匹配除换行符以外任意字符 .后加{n}表示匹配任意n位

  • ’{n,m}‘表示匹配n到m位 ‘{n,}’表示匹配n位及n位以上

  • ‘[^]’表示不匹配本内容

  • ‘.+？’匹配任意字符一次到多次，?为抑制贪婪匹配，匹配到第一个就停止

    转义字符

  • ‘\’表示转义

  • ‘\d’表示匹配数字（只是一位 下同）

  • ‘\w’类似等价于表示匹配字母数字下划线

  • ‘\W’类似等价于表示匹配非字母数字下划线

  • ‘\D’表示匹配非数字

python中使用正则表达式

  • 导入re模块

    match函数

    match(正则表达式，字符串, 修饰符)

    扫描字符串，从头开始匹配

    search函数

    search(正则表达式，字符串, 修饰符)

    扫描字符串中所有符合正则表达式的字符串，并返回第一个符合的子字符串

    findall函数

    findall(正则表达式，字符串, 修饰符)

    扫描字符串中所有符合正则表达式的字符串，并返回所有符合规则的子字符串，把匹配到的内容组成列表

    修饰符

    re.I 使匹配对大小写不敏感

    re.M 多行匹配，影响^和$

    group函数

    split函数

    split(pattern, string, maxsplit, flags)

    表达式，字符串，次数，标志位

    sub函数

    sub(pattern, repl, string, count, flags)

    表达式， 替换的字符串, 被替换的字符串， 次数， 标志位

subn函数

sub(pattern, repl, string, count, flags)

表达式， 替换的字符串, 被替换的字符串， 次数， 标志位

返回由替换后的内容和替换的次数组成的元组
