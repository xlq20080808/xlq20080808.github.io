Python-异常和IO

异常处理

  • try里边出现异常 直接跳出执行except然后执行下面代码 try里边剩余的语句不执行
  • 如果不加except或者except后加的类型与异常的类型不同 则执行完finally报错后就停止不执行下面代码

断言

  • 在程序发生异常之前，先做一个断言操作 多用于程序调试阶段

assert 1 == 2 , "错误的表达式"

  • 当表达式返回时False时 抛出断言的错误，输出自定义的错误信息

IO流

读模式

  • 当文件不存在时，会抛出异常

写模式

追加写模式

  • 文件内容基础上继续在尾部追加内容

读写模式

r+

  • 读模式的基础上覆盖写 要求文件必须存在

w+

  • 写模式下增加读操作，文件不存在时创建

二进制读写

  • rb和wb ，以二进制形式读写，常用于音频 图片

文件和目录

  • 创建目录os.mkdir

序列化

  • 将对象存到文件当中

反序列化

json序列化

  • 为使其他编程语言识别

with as打开文件

  • 不需要关闭IO流
