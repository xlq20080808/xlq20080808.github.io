Python-for循环迭代器

for循环

  • range (5) 0,1,2,3,4
  • range(5,10) 5,6,7,8,9
  • range(10, 5, -1) 10,9,8,7,6
  • 无i++ 用i += 1

迭代器

  • List 字符串 元组可迭代

iter 迭代

sorted 迭代

  • 排序后迭代,针对可比较元素序列

reversed 反转迭代

  • 完成倒序输出

zip 迭代

  • 可以同时迭代多个序列
  • 如果迭代的序列长度不一致，迭代次数是最小的那个

enumerate 迭代

  • 枚举迭代器 可以迭代出索引位置
  • 迭代出索引位置

列表解析式

  • 直接转换为列表
  • 将两个数量相同的列表 对应位置组成元组最终组成元组列表 如· [1, 2, 3]和 [4, 5, 6]组成[(1, 4), (2, 5), (3, 6)]
