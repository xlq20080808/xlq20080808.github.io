Python-函数

函数

基本格式

  • 函数里没有写return也可以用变量进行接收
  • python函数中任何函数都有返回值 当函数定义时有return 则函数返回值就是return所返回的内容 当函数定义时没定义return 则默认返回None
  • 一个函数中没有方法体

传参

  • 位置参数 按照形参的位置传值
  • 关键字参数 能和位置参数结合使用 可以传部分作为关键字 但关键字参数不能在位置参数之前 不允许重复赋值
  • 默认参数 定义函数时指定某些默认值，调用时可以通过位置参数和关键字参数对默认参数赋值
  • 非关键字可变参数 func(a, b, *k)k 参数的数量不一定 按照元组的形式来接收对数据类型没要求 传参以位置形式传入 但*k必须在参数列表最后
  • 关键字可变参数func(a, b, **k)传参时func(1, 2, c=3, d=4)

解包裹参数

  • 在传递元组时，让元组的每一个元素对应一个位置参数，需要解包裹传递
  • 在传递词典字典时，让词典的每个键值对作为一个关键字参数传递给函数

变量

  • 全局变量与局部变量
  • 函数中想要使用全局变量 可直接访问 比如输出，但不能修改
  • 想赋值或修改 则用 global关键字
  • 函数内操作全局变量
  • 函数内的函数调用外部函数的变量
