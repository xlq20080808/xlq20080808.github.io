Python-基础

Python基础

输入输出

  • 逗号输出空格
  • Python的整数、浮点数都没有大小限制

ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符

ord(A) = 65 chr(65) = A

  • input()输入后得到的是字符串

​ python解释器认为import的文件是可重复利用的模块或对象，此时会把import的文件”编译”为pyc文件，

​ python3会把pyc文件放置pycache文件夹中用于重复利用

编码规范

  • 逗号冒号 前没空格 后加空格
  • 冒号当结尾符不用加空格
  • 运算符两侧都加空格
  • 赋值符号 = 号作为函数参数定义和传参赋值时两边都不加空格
  • 空格缩进 一般4个空格 4个空格和table缩进不同级会报错

数据类型

布尔类型

  • True 是真 False假

complex 复数类型

  • 如 a = 2 + 3j 可以通过real和image属性获取实部和虚部的值 conjugete()获得共轭复数
  • 当几种数字类型进行运算时结果的类型按照 complex>float>int进行排序

字符串

  • 单行字符串用单引号和双引号都可以 ，引号里边有需要有引号时则单双引号交替使用
  • 多行字符串用三个单引号或者双引号，为和多行注释区分 注释时引号和内容在不同行写
  • print("c:\\news") 相当于使用row print(r"c:\news") r抑制转义，不用加 \进行转义

​ \b 退格 print("hello\bworld") 输出“hellworld”

  • 格式化字符串

​ print("hello%s"%("world"))输出helloworld %s为占位符 显示的是”%()”里边的内容 还有%d等占位符

  • 字符串运算

​ 字符串可以用 [ ]索引 赋值完后不可改变·a[0] = 'a'报错

  • 切片

    a[0: 3]得到‘hel’ 从0到3但不包括3 当冒号后超过最大 取最大

​ a[0: 5: 2]先取a[0: 5] 得hello ‘2’为步数 每2字节取一个 得hlo

​ a[5: 0：-1]倒着取 从5到0但不包括0

  • 字符串方法
  1. index 和find都是找字符串里的第一个相同字符并返回索引 找不到时index抛出异常 find返回-1 index里可以放子字符串 返回的是第一个子字符串第一个字母的索引
  2. replace 替换字符串 由于字符串不可变 返回一个新的字符串,与原字符串无关
  3. count 返回某个字符或字符串的数量
  4. split 以某一字符串为分割,舍去此字符串,当这个字符串为结尾，得到的列表最后一个为空
  5. len 返回长度,对有序列都可用,列表也可,列表以逗号为分割,返回有几个元素

列表

  • 一组有序数据 属于可变对象 数据可包含各种数据类型 ，可重复 可len 可切片 可*2 两两相加等 可变
  • 列表方法
  1. index 求索引 没有find方法
  2. append 追加 追加到最后
  3. insert 插入 insert(2,’nn’)插入到索引2所在位置 其他向后移动
  4. count 求数量 count(‘nnn’)
  5. reverse 反转 改变原列表
  6. remove 删除对应元素 且包括列表中所有的重复的元素remove(‘nnn’)没有则抛出异常
  7. del 直接删除列表中对应位置的元素
  8. pop 弹出 当成堆栈 从后弹出 （先进后出）

元组

  • 不可变,不经常变动的数据放元组 变动的放列表
  • 如果只有一个元素 如 t = (40)

​ 则把t当成普通数据类型 就没有元组的操作 想定义成元组 则定义为（40,）是不可变序列对象 里边的元素是不

​ 能直接修改的 如果里边的元素是一个可变的 比如元素是一个列表 可以更改可变对象里边的东西 但还是不能

​ 变这个元素外部,如不能把元组里的列表替换成其他类型

  • t=(1, 2, [1,2,3])

​ t[2] [2] =5 可以进行修改 t[2] = "abc"则不能修改

  • 元组会缓存在解释器 而列表不行
  • 所有序列对象都能进行 合并 索引 重复 切片 是否存在操作（判断是否存在 用 in 和 not in，合并和重复等生成新元组与原元组无关）

字典

  • 可变的
  • 用大括号包含起来的多个键值对的组合 键值对之间用，隔开
  • d = {1： 2， “a”: 1, "b": 2}也可用 dice({})来定义 或 dict(zip([1,"a", "b"] [2,1,2]))把两个列表定义成字典{1:2, "a": 1, "b": 2}
  • ==判断是否相等
  • 通过键名获取值 d[“a”] 得1 如果定义时有重复的键名,后定义的覆盖先定义的 一般不允许定义重复键值名
  • 字典方法
  1. len 返回有几个键值对
  2. 修改 d[1] = 12
  3. 增加 d["d"] = 5
  4. 合并 update 字典之间的合并d4.update(d5)
  5. 删除 del
  6. pop 需要键名 将键名对应的函数弹出 相当于del有返回值 d.pop(1)
  7. popitem 弹出最后的键值对 以元组的形式返回 无需键名
  8. in运算 判断键名是否存在于该字典
  9. items以元组方式返回键值对组成的列表 keys返回键值列表 values返回值列表

运算符

  • /真除,得到的是小数 //整除
  • and or not 与或非

3.ssm整合的基础配置

ssm单独开发的时候：单独使用spring，需要applicationContext.xml；单独使用springMVC，需要springMVC.xml

所以：整合的时候，我们还是需要这三个配置文件

3.1 spring的基本配置

  • 一般存放到src同级文件夹config中
  • 创建配置文件applicationContext.xml、springMVC.xml、mybatis.xml配置文件（用maven还有pom.xml)，配置文件的头部约束和命名空间等
  • 如果第一次配置，每个配置文件的官方文档中都带有一个设置好的配置文件，下载即可
3.1.1 简单介绍
  • spring：起到对其他框架以及类的管理和粘合的作用（对象的管理，事务的管理、注解、反射、）粘合springMVC与mybatis。主要思想：注解、反射、面向切面编程
  • springMVC：基于servlet，代替servlet进行前后端的交互，MVC模式进行控制
  • mybatis：主要数据库操作，动态sql语句，二级缓存
3.1.2 配置文件步骤
